Random Website Facts (January 06)

One of the best things about doing the websites the almost-hard-way (running a site, remote hosting – as opposed to self-hosting which is the really-hard-way) is that you can occasionally browse the pretty pictures that they (the hosting company aka luxsci) provide for usage patterns in terms of page hits and Kbytes transferred etc. (Yawn).

I spent a few minutes doing this tonight, but want to draw attention to my favourite section: search terms that hit the website (this includes the three main blogs – bfp, pd, pm – and the main machaggis.net website).

So you can tell many things from this list:

  • danika is the favourite search term
  • we don’t exactly solicit search engine placement: danika is only found twice in January, three times if you count danika marschner and we only got 17 matches overall!
  • someone searching for crazy wedding dancing found us
  • we’ve historically always had occasional hits for the Ivy Guest House in Siem Reap in several formats (see our world travels)
  • I also don’t know why we match to pink canoes and gaitano’s new jersey, think of it as ethnic diversity

And if you’re interested it looks like on Windows derivatives is the favourite browser with bloglines (how-to) second and Firefox-derivates on the Mac third (which is probably me!).

Oh yeah, and bluefacedpixie has more hits than planetdanika. But then, it’s almost midnight, I’m geeking out and Danika is in bed.

The tip of the iceberg

I spent way too long on Sunday doing non-exciting computer things with machaggis.net. I should have been doing a little web project for a friend which is long overdue but instead I had to tackle issues surrounding running out of hosted webspace for machaggis.net. I’m hosted on luxsci.com and only pay for 100Mb of space, which for the first time has started to get tight (mostly due to transient email).

So I spent 3+ hours syncing CVS directories, deleting stuff and generally being paranoid about completely erasing and restoring machaggis.net. If you see anything strange or broken let me know.

As all of this is completely invisible I decided to finish up the fruitless Sunday by tweaking bluefacedpixie. I’ve been getting some comments that planetdanika is ‘nicer’ and ‘cleaner’ than bfp. Some of this is, I’m sure, unjustified and probably just relates to to the use of san-serif fonts, bigger text and cool pics (which I took I must add). But I have succumbed, and admit that bfp was a little bit of a playground for design for a while, so I’ve switched most fonts to sans except for the main stories (serif is meant to be easier to read in the long run, even if it doesn’t seem so at the time, and I like Georgia!). I’ve also pulled the optional formatting for blog this, upped the font size and gutted the sidebar. I also hacked the wordpress calendar so that the current date would appear in bold.

So the tip of the iceberg from today’s boring geekery and hackery is some lipstick on the blog. Enjoy. (Comments welcome).
